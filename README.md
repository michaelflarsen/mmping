# M1Scan - Network Utility

Windows desktop applikation til netværksscanning og IP-konfiguration.

## Funktioner

- **Netværksscanning** — Ping/ARP-scanning, finder enheder i realtid
- **Netværksadaptere** — Vis og nulstil netværksadaptere
- **IP-konfiguration** — Skift mellem DHCP og statisk IP
- **DNS-flush** — Ryd DNS-cache med ét klik

## Krav

- Windows 10 eller nyere
- .NET 8.0 Runtime
- Administratorrettigheder (til IP-konfiguration)

## Byg og kør

```bash
dotnet build
dotnet run
```
